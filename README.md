# Acervo digital

## Instruções para instalação Local

### Clonar repositório
```
git clone https://github.com/Silvestrebat/AcervoDigital.git
```
### Instalar dependências

```
cd AcervoDigital && npm install
```

## Instruções para execução
```
npm start
```
